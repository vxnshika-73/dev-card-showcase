# Countdown Timer

A modern, responsive **Countdown Timer web application** built using  
**HTML, CSS, and Vanilla JavaScript** with light/dark mode support.

This project focuses on clean UI, accurate timing logic, and mobile-first responsiveness.

---

## Features

- Set countdown using hours, minutes, and seconds
- Start, pause, resume, and reset controls
- Live countdown display
- Prevents multiple timers running simultaneously
- Visual indication when timer completes
- Dark mode with smooth theme transition
- Fully responsive (mobile, tablet, desktop)
